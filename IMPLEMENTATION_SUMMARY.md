## 🎓 How to Customize

### Adding Data:
The app is a single HTML file. To customize:
1. Open `competitive_analysis.html` in a text editor
2. Find the content section you want to edit
3. Update text, numbers, or structure
4. Save and refresh in browser

### Changing Colors:
All colors are CSS variables in the style section:
- Primary: `#667eea` (purple-blue)
- Secondary: `#764ba2` (darker purple)
- Success: `#28a745` (green)
- Danger: `#dc3545` (red)
- Warning: `#ffc107` (yellow)

### Adding Sections:
To add a new section:
1. Create new `<section>` element
2. Add navigation tab in header
3. Add JavaScript click handler
